# USR Bananium Rush

This is a 3-weeks school project made in group of 3 devs.
The goal was to make our own game with no restrictions. We used our custom engine "FSEngine" made by the same team of 3 in 4 weeks. 
For this project, we also had one week of pre-prod.

To launch the project : 
- bin -> make
- return to the folder
- ide -> launch the .sln
